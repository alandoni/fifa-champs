# fifa-champs

#On OSX

Dependencies:
node
brew

#Run Backend

install mongo
```brew install mongodb
```

install dependencies
```cd fifa-champs
cd Backend
npm install
```

run mongo
```mkdir -p /data/db
./run_mongo.sh
```

run server
```npm start
```

#Run Backend
```
cd ../Frontend
npm install
sudo npm install -g angular-cli
ng serve
```
